# Dashboard Overview

The Fern-OS Dashboard provides a real-time view into your distributed orchestration engine. It allows you to monitor system health, track workflow execution, and manage worker nodes through an intuitive, glassmorphic interface.

## Quick Navigation

- [**Main Dashboard**](dashboard.md): High-level system metrics and recent activity.
- [**Workflows**](workflows.md): Visualize and track DAG-based workflow executions.
- [**Jobs**](jobs.md): Detailed history and logs for individual task executions.
- [**Workers**](workers.md): Monitor worker node status, health, and resource usage.

---

## Accessing the UI

By default, the dashboard is accessible at `http://localhost` when running via Docker. For custom deployments, refer to the [Setup Guide](../setup/index.md).
